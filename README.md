# Activity Calculator

Проект для расчета активности пользователей на основе данных Яндекс.Метрики.

## Файлы проекта

- `metrika_activity.py` - основной модуль для расчета активности
- `analyze_raw_data.py` - анализ сырых данных
- `verify_activity_score.py` - проверка расчета активности
- `test.py` - тестовый скрипт

## Данные

- `activity_2025-06-25.json` - рассчитанная активность
- `raw_hits_2025-06-25.json` - сырые данные о хитах
- `raw_visits_2025-06-25.json` - сырые данные о визитах
- `etalon_visits.json` - эталонные данные для проверки
